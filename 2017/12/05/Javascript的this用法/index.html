this总是指向调用函数的那个对象

纯粹的函数调用

this指向window

作为对象方法的调用

this 指向这个上级对象

作为构造函数调用

所谓构造函数，就是通过这个函数生成一个新对象（object）。这时，this就指这个新对象。

apply调用

pply()是函数对象的一个方法，它的作用是改变函数的调用对象，它的第一个参数就表示改变后的调用这个函数的对象。因此，this指的就是这第一个参数。

apply()的参数为空时，默认调用全局对象。因此，这时的运行结果为0，证明this指的是全局对象。

如果把最后一行代码修改为
　　o.m.apply(o); //1

运行结果就变成了1，证明了这时this代表的是对象o。
